# Voting App

A simple Voting Application built using **Node.js, Express.js, MongoDB, JWT, and REST APIs**. It allows users to register, log in securely, vote for a candidate, and view election results.

## Features

* User Registration & Login
* JWT Authentication
* Secure Password Hashing (bcrypt)
* Vote for Candidates
* View Results
* Protected Routes
* MongoDB Integration
* REST APIs
* API Testing with Postman

## Tech Stack

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT
* bcrypt
* Postman

## Installation

```bash
npm install
npm start
```

## API Endpoints

* `POST /signup`
* `POST /login`
* `GET /candidates`
* `POST /vote/:id`
* `GET /results`
